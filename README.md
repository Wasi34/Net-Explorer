# Net-Explorer
Net Explorer is a web browser like Microsoft Edge.

GUI toolkit: JavaFX

Features:

• Home Button

• Address bar

• Bookmark buttons

• Navigation buttons

• Refresh button

• History button

Software Used: 

• Netbeans 8.2

• Scene Builder

• XAMPP 8.0.3
