# Net-Explorer
Net Explorer is a web browser like Microsoft Edge.

Features:
<br/>
• Home Button
• Address bar
• Bookmark buttons
• Navigation buttons
• Refresh button
• History button
<br/>

GUI toolkit: JavaFX

<br/>

Software Used: 

<br/>
• Netbeans 8.2
• Scene Builder
• XAMPP 8.0.3
